# 📚 Descripción General

Este repositorio contiene la solución al **Taller de Ciclos** de la asignatura Lógica de Programación.

El objetivo del taller es desarrollar varios programas en Python utilizando estructuras repetitivas (`while`, `for`), validación de datos, manejo de excepciones y funciones.

El taller se compone de **varios programas independientes**, además de un **programa principal con menú infinito** que integra múltiples funcionalidades.

---

# 🎯 Objetivos del Taller

- Aplicar estructuras repetitivas
- Validar entradas del usuario
- Manejar excepciones para evitar errores en ejecución
- Implementar funciones reutilizables
- Resolver problemas matemáticos usando ciclos
- Mejorar la interacción en consola mediante ASCII Art

---

# 🧾 Programas Desarrollados

## 🔹 Programa 1 — Serie de Fibonacci
Programa que calcula el término n de la serie de Fibonacci utilizando ciclos.

---

## 🔹 Programa 2 — Número Primo n-ésimo
Programa que calcula el número primo correspondiente a la posición indicada por el usuario.

---

## 🔹 Programa 3 — Raíces de una Ecuación Cuadrática
Programa que calcula las raíces de la ecuación cuadrática:

ax² + bx + c = 0

El sistema contempla:
- Raíces reales diferentes  
- Raíces reales iguales  
- Raíces complejas  

---

## 🔹 Programa 4 — Promedio de n Números
Programa que permite ingresar números decimales y calcular su promedio.

---

## 🔹 Programa 5 — Juego Adivinar Número 🎲
Juego donde el usuario debe adivinar un número aleatorio dentro de un rango definido.

El sistema proporciona pistas hasta acertar.

---

## 🔹 Programa 6 — Tabla de Multiplicar 📊
Programa que muestra la tabla de multiplicar del número indicado por el usuario.

---

# ⭐ Programa Principal — MENÚ INFINITO

El programa principal integra múltiples ejercicios en un único sistema interactivo.

El programa se ejecuta continuamente hasta que el usuario selecciona la opción de salir.

# ⚠️ Validaciones Implementadas

El sistema valida y controla:

- Entradas no numéricas
- Valores negativos cuando no aplican
- Manejo de excepciones con `try / except`
- Prevención de cierres inesperados del programa

---

# 🔄 Flujo del Programa Principal

Inicio → Mostrar menú → Seleccionar opción → Ejecutar función → Regresar al menú → Salir
